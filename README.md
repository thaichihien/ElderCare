## Back end

### Hướng dẫn chạy server
- Mở cmd, vào thư mục server:
```
cd server
```
- Tải tất cả dependencies:
```
npm i
```
- Chạy server (chế độ dev):
```
npm run start:dev
```
- Khi xuất hiện dòng bên dưới thì coi như thành công
```
server is ready at http://localhost:3000
```
